# 项目设计：AI 解决方案/FDE 助手 + Agent Harness Layer

## 项目定位

面向 AI 解决方案/FDE 岗位，模拟从客户需求到方案交付的完整链路。

## 输入

- 客户会议纪要
- 需求描述
- 招标文件片段
- 现有系统说明
- 预算和上线周期约束

## 输出

- 客户行业和业务背景
- 核心痛点
- 系统约束
- 可用 AI 能力
- 方案大纲
- 实施计划
- 风险清单
- 待确认问题
- 人工确认节点

## Agent Harness Layer

- Agent loop：计划、工具调用、观察、下一步决策、终止条件。
- Tool registry：可用工具、参数 schema、权限边界。
- State：客户信息、约束、已确认问题、方案版本。
- Guardrails：敏感信息、预算、权限、人工确认节点。
- Trace：每一步为什么调用工具、工具返回了什么、是否失败。
- Recovery：工具失败、信息不足、循环调用、超时后的处理策略。

## 面试价值

这个项目能体现你不是只会做后台管理系统，也不是只会调模型，而是能把客户现场里的模糊需求转成可交付方案。Agent Harness 是重点：它说明你理解企业 Agent 不只是“能自动生成方案”，还要能追踪、审核、恢复和控制风险。
