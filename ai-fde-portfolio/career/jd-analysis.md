## 高频能力归纳

技术：

- Python/FastAPI 或 Java/Spring Boot 集成能力
- 大模型 API、Token、计费、错误处理
- RAG、Agent、工作流、工具调用
- Eval Harness、RAG Evaluation Harness、Agent Harness、trace、回归测试、CI gate、失败归因
- 日志、监控、评测、成本控制
- AI 辅助开发、Python/FastAPI、API 原型、测试、debug、代码理解
- Agent 生命周期管理、权限控制、运行监控、灰度发布、API/SDK 接入文档
- 标注体系、解决率、分流比、知识库质量闭环

产品/方案：

- 客户需求抽象
- 方案设计
- 指标拆解
- 交付风险管理
- 业务价值表达
- 评测集管理、Bad Case 管理、生产 trace 回流、发布门禁、人工标注流程
- 平台共性能力抽象：把不同业务场景沉淀成 Agent 元数据、技能描述、工具协议和治理能力

行业：

- 制造业

- 政企数字化
- 企业服务
- 教育/医疗可作为备选

## 本周市场结论

快照日期：2026-05-28。详见 `market-snapshot-2026-05-28-ai-pm-jd.md`。

- 重庆本地已经出现 Agent 平台产品机会，但样本少、薪资跨度大；12K 仍是下一阶段合理底线。
- 成都样本更强调 Agent 从 0 到 1、工具调用、RAG、评测指标和失败案例分析，可作为相邻市场参考。
- AI Builder 技术能力不能删，重庆 AI 应用工程样本显示 RAG、Agent、多 Agent、Prompt 和框架选型有更高薪资锚点。
- 下一步简历应强化：LLM Gateway、Eval Harness、Agent 生命周期、RAG 质量治理、AI 辅助开发、测试/eval/trace。
