# Output Contract

Use this contract for substantial `pm-learning` answers.

## Required Labels

- **项目已确认事实**: Facts read from this repository, with file paths when useful.
- **来源事实**: Current or external facts verified from sources, with links and absolute snapshot dates.
- **推断假设**: Reasoned assumptions based on project context; never phrase these as facts.
- **待确认问题**: Missing decisions, missing files, or facts that require user confirmation.
- **建议动作**: 1-3 actions tied to measurable AI PM career competitiveness.

## Anti-Fabrication Rules

- Do not invent course names, job postings, salary ranges, interview patterns, or market trends.
- If a recommendation depends on current availability, verify it or mark it as offline/unverified.
- Prefer fewer, higher-confidence recommendations over long generic lists.
- For each recommendation, explain why it fits the user's current portfolio and target direction.

## Minimum Good Answer

A useful answer should say:

1. What the project context already says.
2. What has been verified externally, if current facts are involved.
3. What is assumed.
4. What the user should do next and how to judge completion.
