# Skill Routing

Use the most specific skill that can produce the artifact.

## Routes

- `$pm-learning-roadmap`: learning paths, course/resource recommendations, weekly/monthly plans, study task slicing.
- `$pm-learning-evaluator`: weekly review, scorecards, learning supervision, accountability, reminder setup.
- `$pm-learning-market-research`: job descriptions, hiring market, salary, interview requirements, target-company scan.
- `$pm-learning-career-packaging`: resume, portfolio, project case studies, interview scripts, career positioning.

## Multi-Step Requests

For broad requests, sequence the skills:

1. Market research: identify current target-role requirements.
2. Roadmap: turn gaps into learning tasks and project outputs.
3. Evaluator: define scoring and supervision cadence.
4. Career packaging: convert outcomes into resume/interview assets.

## Example Requests

- "帮我规划下周学习重点" -> roadmap first, evaluator for acceptance checks.
- "推荐 RAG 评测课程" -> roadmap with source verification.
- "调研重庆 AI 产品经理机会" -> market research with dated sources.
- "把 LLM Gateway 写进简历" -> career packaging.
- "我最近学得乱，帮我收束一下" -> OS, then route based on gaps.
