# Evaluation Rubric

Score each dimension 0-5.

## Dimensions

| Dimension | 0-1 | 2-3 | 4-5 |
| --- | --- | --- | --- |
| Plan completion | No clear progress | Some tasks done | Planned outputs completed or intentionally adjusted |
| Artifact quality | Notes only | Draft artifact exists | Artifact can support portfolio, resume, or interview |
| AI PM reasoning | Generic learning | Some product framing | Clear user, scenario, metric, risk, tradeoff, quality loop |
| Interview/resume transfer | Cannot explain | Has rough wording | Has concise bullets and structured story |
| Market-role fit | Not tied to target roles | Some JD connection | Clearly improves target-role competitiveness |
| Next-step clarity | Vague | Actionable but broad | 1-3 specific next actions with acceptance checks |

## Interpretation

- 24-30: strong week; package output and raise difficulty.
- 16-23: useful week; fix one weak dimension next.
- 8-15: activity exists but conversion is weak; reduce scope and create one visible artifact.
- 0-7: reset plan; choose one output for the next 3-5 days.

## Required Evidence

Look for:

- Updated project docs.
- Running project/demo/test result when relevant.
- Resume bullets or interview scripts.
- JD analysis or market evidence.
- Weekly review notes.
